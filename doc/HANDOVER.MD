## 已知边界

- 自动裁边依赖纸张外缘与背景存在可见亮度或颜色差；纸张与背景完全同色、四边大面积被遮挡时会回退原图，避免错误裁掉正文。
- 微信分享面板必须在真实微信客户端中由用户点击触发，开发者工具只能验证接口编译与调用路径。
- Sass 构建仍有上游 legacy API 与 `@import` 弃用警告，不影响当前生产包。
