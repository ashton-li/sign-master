## 真机复验重点

- 在四个主页面分别点击右上角“···”，确认“发送给朋友”和“分享到朋友圈”均出现，标题与封面正确。
- 在“我的 > 推荐签字大师”中点击“分享给好友”，确认直接打开微信好友选择；点击“分享到朋友圈”后按提示从右上角菜单发布。
- 在“我的签名”打开任一签名预览，点击右上角分享按钮，确认微信显示图片分享菜单且签名图完整、背景透明。
- 从好友消息和朋友圈卡片重新进入小程序，确认落到文件首页且底部导航选中状态正确。

## 已知边界

- 微信不提供由普通页面按钮直接发布朋友圈的接口；本项目开启原生朋友圈菜单并给出明确操作引导，最终发布必须由用户在右上角菜单确认。
- `wx.showShareImageMenu` 依赖较新的微信客户端；旧版本会显示升级提示，不会降级成不可预览的文件附件。
- Sass 构建仍有上游 legacy API 与 `@import` 弃用警告，不影响本次生产产物。
